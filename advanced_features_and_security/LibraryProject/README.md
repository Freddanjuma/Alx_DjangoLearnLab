# LibraryProject

This is my Django project for ALX Django Learn Lab.

## Project Structure

- **manage.py** → A command-line utility that lets you interact with this Django project.
- **settings.py** → Configuration for the Django project (database, installed apps, middleware, etc.).
- **urls.py** → The URL declarations for the project; acts like a “table of contents” for your site.
- **wsgi.py / asgi.py** → Entry points for WSGI/ASGI servers when deploying the project.
- **db.sqlite3** → The default database created by Django.
